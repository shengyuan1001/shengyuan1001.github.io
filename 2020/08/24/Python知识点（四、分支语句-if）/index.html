Python知识点（四、分支语句-if）
Created2020-08-24|Updated2020-08-24
|Post View:

在编程语句中，代码的结构分类：

顺序结构：代码从上往下依次执行

分支结构：代码在执行的过程中根据不同的条件执行不同的代码

循环结构：代码在执行的过程重复执行某段代码

简单if语句【单分支】

语法：

​ if 条件:

​ 语句

说明：

​ a.条件可以是常量，变量或者是表达式

​ b.在Python中，冒号代表代码块的开始

​ c.在Python中，通过缩进区分代码块

​ d.工作原理：要么执行，要么不执行，是否执行取决于条件是否为真

​ e.何为真假?

​ 假：0 0.0 “” None False ,各种空的数据

if-else语句【双分支】

语法：

​ if 条件:

​ 语句1

​ else:

​ 语句2

说明：

​ a.工作原理：实现了二选一的操作，如果条件成立，则执行语句1；如果条件不成立，则执行语句2；不管条件成立与否，两个分支总会执行其中的一个

if-elif-if语句【多分支】

elif——->else if

语法：

​ if 条件1:

​ 语句1

​ elif 条件2：

​ 语句2

​ elif 条件3：

​ 语句3

​ ….

​ else:

​ 语句n

说明：

​ a.多分支实现了多选一的操作【不管其中有多少个条件满足，都只会执行其中的一个分支，从上往下依次判断，只要遇到一个成立的条件，则执行对应的语句，执行完成之后整个if-elif-else语句结束】

​ b.如果所有的条件都不成立，则直接执行else分支

三者之间的区别

嵌套if语句

在一个if语句中定义一个if语句

语法：

​ if 条件1：

​ 语句1

​ if 条件2：

​ 语句2

说明：

​ 工作原理：只有当条件1和条件2同时成立时，语句2才有执行的机会
