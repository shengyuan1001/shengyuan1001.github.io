Python知识点（三、运算符与表达式）
Created2020-08-24|Updated2020-08-24
|Post View:

1.表达式

由操作数和运算符组成的一条语句，被称为表达式

举例：

3 + 5就是一个表达式，3和5就是操作数，+就是运算符

作用：可以进行指定的运算，可以给变量赋值

2.运算符

算术运算符

作用：进行数学运算

分类：+ - *【乘】 / 【除】 %【求余，取模】 //【取整】 **【求幂】

赋值运算符

作用：给变量赋值

分类：

​ 简单赋值运算符：=

​ 复合赋值运算符：+= -= *= /= %= **= //=

注意：一旦在一个表达式中出现赋值运算符，则一定是先运算的是等号右边，然后再给等号左边的变量进行赋值

关系【条件，比较】运算符

作用：比较大小

分类：== != < > <= >=

注意：得到结果肯定为布尔值，成立则结果为True，不成立则结果为False，一般结合if语句或者while语句使用

位运算符【了解】

二进制

&【按位与】 |【按位或】 ^【按位异或】

~【按位取反】 <<【左移】 >>【右移】

真：1 假：0

注意：^,&,|

逻辑运算符

作用：用来做逻辑计算的，用来判断的表达式或者变量，运算得到的结果为布尔值

分类：and【逻辑与】 or【逻辑或】 not【逻辑非】

注意：逻辑运算符可能会出现短路原则【出现在复杂的表达式中，一般会出现在面试题中，在实际开发中，一般逻辑运算符不要超过3个】

成员运算符

注意：成员运算符一般结合数据结构【list,tuple等】使用

in：如果在指定的序列中找到指定的值，则返回True

not in：如果在执行的序列中找不到指定的值，则返回True

身份运算符

和成员运算符类似，运算之后的结果都是布尔值，一般结合if语句使用

is:判断两个变量是否属于同一个实体【对象】

is not：判断两个变量是否不属于同一个实体【对象】

运算符优先级

注意：

​ a.将复杂的问题尽量简单化【分步操作】

​ b.如果出现复杂的表达式，则使用（）提高优先级，控制表达式的执行顺序，否则实现的代码可读性太差，可维护性差
