list列表的简单用法

简单概念

变量：存储数据，参与运算【简单理解：变量是一个容器，但是，一次只能存储一个数据】

可能存在的问题：如果需要存储的数据量大，则占用的内存也会随着增大

解决方案：采用列表

作用：列表相当于是一个容器，区别于变量，同时可以存储多个数据

​ 例如：如果用列表存储5个人的年龄，list1 = [age1,age2,age3,age4,age5]

本质：列表是一种有序的集合

有序：有顺序【数据的存放顺序和在内存中存储的顺序相同】

注意：本身仍然是一个变量，其中存储的是变量

创建

创建列表实际上相当于定义一个列表类型的变量

语法：变量名 = 初始值

​ 列表名 = [数据1，数据2，数据3.。。。。。]

说明：

​ a.使用[]表示创建列表

​ b.列表中存储的数据被称为元素

​ c.一旦将数据存储到列表中，列表会自动给数据进行编号，编号从0开始，该编号被称为索引/下标/角标

​ d.索引的取值范围：0元素的个数- 1【0列表的长度 - 1】

​ e.注意事项：超过索引的范围则会出现索引越界

功能：主要用于遍历容器【遍历：依次访问容器中的每一个元素，获取每个元素值】

工作原理：按照顺序获取列表中的元素，赋值给变量，再执行循环体，如此循环往复，直到列表中的元素全部被获取完

列表生成器

range([start],end[,step]),start和step可以省略

start：开始数字

end：结束数字

step：步长

注意1：start默认为0，step默认为1

注意2:通过range生成的容器并不是列表，如果相当做列表使用，则需要通过list()转换
